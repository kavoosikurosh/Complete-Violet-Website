# Violet Pancreatic Cancer Awareness Foundation

A 501(c)(3) nonprofit dedicated to reducing pancreatic cancer mortality through education, early detection resources, and youth health literacy.

## Website

Live site: [violetaware.org](https://violetaware.org)

## Features

- **Risk Assessment Tool**: Science-backed questionnaire analyzing 8 key risk factors
- **Educational Resources**: Comprehensive library about pancreatic cancer prevention and early detection
- **Community Support**: Resources for patients, survivors, and families

## Tech Stack

- React 18
- Tailwind CSS
- Deployed on Vercel

## Local Development

```bash
npm install
npm start
```

## Deploy to Vercel

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Click "Import Project"
4. Select your GitHub repository
5. Vercel will automatically detect it's a React app and deploy it
